# AgentForgeOS Apps

Applications extend the platform without reaching into engine internals. Each module consumes services and providers while remaining sandboxed from core runtime state.

- **studio/** – Studio interface helpers.
- **builds/** – Build orchestration helpers.
- **research/** – Research dashboards and pipelines.
- **assets/** – Asset catalog and pipeline helpers.
- **deployment/** – Deployment orchestrators.
- **sandbox/** – Experimental or isolated features.
